Facebook ID Retriever

Faceuidd is an easy-to-use online tool to grab ID from Facebook profiles, groups or posts. Enter the Facebook link and get the ID quickly..

Please wait 5-30 seconds for the system to retrieve the ID

How to use

  1. Copy the Facebook link of the profile, group, or post you want to retrieve the ID for.
  2. Paste the link into the input field above.
  3. Click on the "Get ID Now" button.
  4. Wait for a few seconds and the ID will be displayed in a modal.
